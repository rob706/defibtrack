# Defib Track

This app is designed to help with tracking Defibrilators (AED's) and thier associated components

It is currently in an MVP state and is being developed with additional functionality in an agile way

## Tech Stack

- PHP
- MySQL

## Installation

1. Download a copy of the code
2. Add database onto MySQL Server
3. Run Database setup script (in folder)
4. Complete the Config File
5. Give Write permissions for the images folder
